#RDF/XML and N3
##A light introduction to their syntax

I was tasked with creating a nice introduction on how to write rdf in rdf/xml and Notation3. Thankfully I found [reveal.js](https://github.com/hakimel/reveal.js/) which made it easy to create some slides by sourcing knowledge from w3.org regarding the [rdf syntax grammar](http://www.w3.org/TR/rdf-syntax-grammar/).

The Notation3 was created using this [RDF validator](http://www.rdfabout.com/demo/validator/).
