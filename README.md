#RDF/XML and N3
##A light introduction to their syntax

I was tasked with creating a nice introduction on how to write rdf in rdf/xml and Notation3. Thankfully I found [reveal.js](https://github.com/hakimel/reveal.js/) which made it easy to create some slides based w3.org: [rdf syntax grammar](http://www.w3.org/TR/rdf-syntax-grammar/). The Notation3 was created using this [RDF validator](http://www.rdfabout.com/demo/validator/).

To read this set of slides, [download a zip of the project](https://github.com/nilsnh/rdf-n3-syntax-slides/archive/master.zip) and click index.html.
